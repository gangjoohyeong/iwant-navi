# iwant-navi

Next.js 16 + TypeScript + Tailwind CSS 4 기본 구성 프로젝트입니다.

## 시작하기

```bash
pnpm install
pnpm dev
```

## 스크립트

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
pnpm typecheck
```

## 폴더 구조

```text
src/
  app/         # page layer
  feature/     # feature layer
  gateway/     # server action api layer
  components/  # ui layer
  hook/
  lib/
  provider/
  utils/
  types/
```
